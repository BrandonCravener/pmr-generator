## Features

- Saves to file
- Automatic copy

---

## Installation

Install this project by running:

1. `$ git clone https://github.com/BrandonCravener/pmr-generator`
2. `$ cd pmr-generator`
3. `$ yarn`

Run with `$ ng serve`
